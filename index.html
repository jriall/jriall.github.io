Projects

A selection of personal projects built to demonstrate and hone my develompment skills.

  • Portfolio Website

    This portfolio website is built using vanilla Javascript, Sass, npm and Gulp as a taskrunner. The website is served via a custom HTTPS domain on Github Pages with Cloudflare.

  • Capital Cities Quiz

    An interactive quiz game which tests your knowledge of capital cities around the world. Built using React and Redux, with Redux Thunk and Redux MiddleWare used to make a async requests to a remote database hosted on Firebase.

  • To Do List App

    A to do list app built with React, the React Transition Group Library, Sass and bootstrapped with Create React App.

  • Interactive Drumkit

    An interactive drumkit/soundboard allowing you to play a set of sounds. Built using React and Create React App, the app allows you to play various sounds, adjust volume, turn the game on and off, as well as change the sound set entirely.

  • Simon Game

    A recreation of a classic Simon memory game, testing your ability to memorise and replicate an increasing series of color combinations.Built with HTML, CSS and Javascript with jQuery.

  • RGB Color Game

    A game which tests your ability to determine colours from either Hex or RGB colour codes. Built using HTMl, CSS and Javascript with jQuery.

  • Pomodoro Clock

    A pomodoro clock which allows you to set work and break intervals to help with productivity. The clock allows for pausing, resetting, changing the length of the intervals and plays a sound and changes color on completion of an interval. Built using HTMl, CSS and Javascript with jQuery.

  • Javascript Calculator

    A relication of a standard tabletop calculator built using HTMl, CSS and Javascript with jQuery.

  • Noughts and Crosses

    A noughts and crosses game where a player can play against a (currently quite dumb!) computer opponent, with the game able to track the cumulative scores in the session. Built using HTMl, CSS and Javascript with jQuery.

  • Cycling Power Calculator

    A tool for cyclists to input their power data and determine their power to weight ratio at various important duration, to see how they compare against various competitive levels within the sport. Built using HTMl, CSS and Javascript with jQuery.

  • Wikipedia Viewer

    An app which takes a user search input and displays partial data pulled from Wikipedia via a call to the Wikipedia API.

  • Weather Viewer

    A web app which takes a users location using the IP-API and displays their local weather to them via an API request to Open Weather.

  • Random Quote Generator

    Displays a random quote to the user, allows them to load new quotes as well as tweet their favourite quotes. Quotes are loaded from forismatic.com's API.
